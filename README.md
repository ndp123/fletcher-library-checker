# Fletcher Price Checker
Opens multiple tabs for bookfinder.com, ebay.com and ebay.com prefiltered to sold items on input ISBN or title.

## Installation

1. Download .zip from this repository

2. Unzip locally (in Windows you can right click and 'Extract here'

3. In Google Chrome, go to settings (the three dots in the upper right corner) >> extensions >> manage extensions

4. Turn on the developer options toggle in the upper right.

5. Click 'Load Unpacked' and select the unzipped folder
